#Music App:ICAI
## Getting Started
### Prerequisites
- JDK-18
- MYSQL 8.0.32
- Tomcat 10.0.27

*Before you start*
- In MYSQL have a database named musicapp
- Go to META-INF.persistence.xml, change value of line 11 & 12

###Services
**Admin:** Can see, edit, create, and delete all data.  
**Users:** Can see & edit their own data. And listen to music.


###Instructions
- Step 1:
    -Configure Tomcat 
- Step 2:
    - Run Tomcat
